## 🎯 Codebase Patterns

### Data Fetching
- Use `unstable_cache` for server-side caching
- Use TanStack Query for client-side data
- Always include error boundaries

### Forms
- React Hook Form + Zod for all forms
- Server actions for form submission
- Optimistic updates where appropriate

### Components
- Use `cn()` utility for conditional classes
- Composition pattern over prop drilling
- Loading skeletons for all async content

### Database
- Always use transactions for multi-step operations
- Soft delete pattern: `deletedAt` timestamp
- Include `createdAt`, `updatedAt` on all tables

### Auth
- Check session in server components: `auth()`
- Protect API routes with middleware
- Role-based access in server actions
